# VIDEOCLUB - CRUD WITH REACT :clapper:

Interface where you can read, create, modify and delete movies that are already registered in the database.

## Features :star:
* Create a new movie.
* Delete a movie.
* Edit a movie.
* Connection to the database in MoongoDB.
